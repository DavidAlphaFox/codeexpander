# 入门

## 设置您的帐户

1. 创建 [Github](https://github.com/join?source=experiment-header-dropdowns-home) 账号.
2. [下载](https://github.com/oncework/codeexpander/releases) 该应用. 

此时，您可以购买软件激活码或继续使用免费模式.
