# 安装

### 安装

对于 windows 用户:

> 您只需配置相关的Python环境变量即可完成安装。

对于 mac 用户:

首次启动软件时，软件将提示如下:

![](../.gitbook/assets/yin-si-an-zhuang.png)

打开系统首选项并选中 `CodeExpander`

![](../.gitbook/assets/yin-si-an-zhuang-2.png)

{% hint style="info" %}
提示：请在设置后重启软件，否则无法正常运行软件。
{% endhint %}

### 编译功能环境支持

```text
$ node -v
$ sh
$ python -v
```

{% hint style="info" %}
如果缺少其中一些命令，请自行下载并安装。
{% endhint %}
